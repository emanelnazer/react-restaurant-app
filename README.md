 # React Restaurant App

This project is a UI design for a restaurant website built with React, Vite, and Tailwind CSS.  
It focuses on the frontend interface only (not a full website with backend).

## Features
- Responsive design
- Dark/Light mode
- Animated sections using AOS
- Image slider using react-slick

## Tech Stack
- React
- Vite
- Tailwind CSS
- React Icons
- AOS for animations
- React Slick for slider
